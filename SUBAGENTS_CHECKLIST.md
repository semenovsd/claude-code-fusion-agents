# Чек-лист для сравнения и оценки Claude Code Subagents

## Методология оценки

Каждый критерий оценивается по шкале от 1 до 10, где:
- **1-3**: Плохо/отсутствует
- **4-6**: Удовлетворительно/базовый уровень
- **7-8**: Хорошо/выше среднего
- **9-10**: Отлично/лучшие практики

## Итоговая оценка

### Взвешенные оценки:
1. Качество промптов: ___/30 × 0.30 = ___
2. Эффективность: ___/25 × 0.25 = ___
3. Функциональность: ___/20 × 0.20 = ___
4. Документация: ___/15 × 0.15 = ___
5. Инновации: ___/10 × 0.10 = ___

**ИТОГОВАЯ ОЦЕНКА: ___/100**

### Рекомендации:
- **90-100**: Отличный агент, рекомендуется к использованию
- **80-89**: Хороший агент, можно использовать с небольшими улучшениями
- **70-79**: Удовлетворительный агент, требует улучшений
- **<70**: Требует значительных улучшений перед использованием
